Project Name: Scooter Delivery

Description:
This is a web-based system that allows customers to send an order or request an order
online integrated with WhatsApp for easy communication and order tracking

Technologies Used:
- HTML
- CSS
- JavaScript
- PHP
- MySQL (XAMPP)

How to Run:
1. Install XAMPP
2. Start Apache and MySQL
3. Import the database from /database/kasi_courier
4. Open http://localhost/kasi_courier/index.html
